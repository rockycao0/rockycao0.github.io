# 定时提醒与自动关机工具使用说明

## 功能特点
- ✅ 后台挂载运行，不影响正常使用
- ✅ 每半小时强制弹窗提醒（时间可配置）
- ✅ 弹窗文字可通过配置文件或命令行自定义
- ✅ 可配置每日强制关机时间
- ✅ 所有配置持久化保存，重启后依然有效

## 快速开始

### 1. 生成默认配置文件
```bash
python reminder.py --config
```
运行后会在当前目录生成 `config.json` 配置文件，默认配置：
```json
{
    "reminder_text": "该休息一下啦，活动活动身体~",
    "reminder_interval": 30,
    "shutdown_time": null,
    "popup_title": "温馨提醒"
}
```

### 2. 直接启动服务
```bash
python reminder.py --run
```
服务会挂载到后台运行，按照默认配置每30分钟弹出提醒。

## 配置方法

### 方法一：直接修改配置文件
编辑 `config.json` 文件，修改对应参数：
- `reminder_text`：弹窗显示的文字内容
- `reminder_interval`：提醒间隔，单位分钟，默认30
- `shutdown_time`：自动关机时间，格式为 "HH:MM"，例如 "22:30"，设置为null则不自动关机
- `popup_title`：弹窗标题

### 方法二：通过命令行配置
#### 设置提醒文字
```bash
python reminder.py --set-text "已经坐了半小时了，站起来走两步~"
```

#### 设置提醒间隔（分钟）
```bash
# 设置为每20分钟提醒一次
python reminder.py --set-interval 20
```

#### 设置自动关机时间
```bash
# 设置为每天23:00自动关机
python reminder.py --set-shutdown 23:00

# 取消自动关机
python reminder.py --set-shutdown none
```

#### 测试弹窗功能
```bash
python reminder.py --test-popup
```

## 后台运行技巧

### Windows系统后台挂载（关闭终端也不停止）
1. 按下Win+R，输入`shell:startup`打开启动文件夹
2. 在启动文件夹中创建一个`reminder.bat`文件，内容：
   ```batch
   @echo off
   python C:\路径\到\reminder.py --run
   ```
   （把路径替换为实际的reminder.py所在路径）
3. 下次开机时会自动后台启动服务

### 临时后台运行（不占用终端）
```bash
# 使用start命令后台运行
start /B python reminder.py --run
```

## 停止服务
- 如果服务在前台运行，直接按Ctrl+C停止
- 如果已经后台运行，可以打开任务管理器，找到python进程结束即可

## 注意事项
1. 自动关机功能会提前1分钟弹出提醒，然后执行关机，有充足时间保存工作
2. 弹窗是系统级强制弹窗，会置顶显示，不会被其他窗口遮挡
3. 所有配置修改后需要重启服务才能生效
4. 建议将脚本放在非中文路径下，避免出现编码问题
